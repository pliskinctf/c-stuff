# 🚀 Systems Programming Portfolio

Welcome to my C development repository. 

### 🧠 About This Project
I am building a foundational understanding of computer systems, memory management, and low-level architecture by writing C. My goal is to strip away the "magic" of high-level languages and understand exactly how software interacts with the CPU and RAM.

### 🛠️ Technology Stack

- **Compiler:** GCC
- **Environment:** Linux / Ubuntu
- **Tools:** Git, GDB, Make
